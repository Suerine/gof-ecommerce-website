## TL;DR

GOF is a full-stack e-commerce platform where customers can browse and purchase jerseys, boots, and footballs inspired by some of the world's greatest footballers. The platform focuses on delivering a modern shopping experience with a clean interface and scalable backend architecture.
Built with **React, Tailwind CSS, Node.js, Express, and MongoDB** it enables users to browse products, add items to a cart, and manage user authentication. The project demonstrates building a responsive ecommerce interface while integrating it with a RESTful backend API and database. Through this project, I strengthened my skills in **frontend-backend integration, API development, authentication, and scalable ecommerce architecture**.

---

# 🛒 GOF Ecommerce Platform

A full-stack **Ecommerce Web Application** that allows users to browse products, view detailed product pages, manage a shopping cart, and authenticate their accounts.  
The platform is designed with a **modern responsive interface and scalable backend architecture**, demonstrating the core functionality of a real-world online store.

This project focuses on **full-stack development, REST API integration, authentication systems, and modern frontend architecture**.

# 🚀 Getting Started

### 1. Clone the repository


git clone https://github.com/yourusername/gof-ecommerce.git


### 2. Navigate into the project


cd gof-ecommerce


### 3. Install backend dependencies


cd backend
npm install


### 4. Install frontend dependencies


cd ../frontend
npm install


### 5. Setup environment variables

Create a `.env` file in the backend directory.

Example:


PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


### 6. Start backend server


npm run dev


Backend runs at:


http://localhost:5000


### 7. Start frontend


npm run dev


Frontend runs at:


http://localhost:5173
